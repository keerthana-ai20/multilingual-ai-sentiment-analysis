# Multilingual AI Sentiment Analysis

AI-powered multilingual customer feedback sentiment analysis system using Transformer models and Flask.

## Features

- Real-time sentiment analysis
- Multilingual NLP support
- Tamil/Hindi/English sentiment detection
- Confidence score prediction
- Interactive dashboard
- Word cloud visualization
- Transformer-based AI model
- Flask web application

## Technologies Used

- Python
- Flask
- Transformers
- BERT
- NLP
- Chart.js
- HTML/CSS/JavaScript

```bash
